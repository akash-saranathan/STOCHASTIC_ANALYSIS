# ForesightLA: Crime Data Probabilistic Analysis in Los Angeles

This project analyzes crime data in Los Angeles using statistical and probabilistic techniques such as **Markov Inequality, Chebyshev Inequality, and Bayes' Theorem**. The goal is to uncover patterns, estimate probabilities, and derive actionable insights for crime trend analysis.

---

## Project Highlights

- Applied **Markov and Chebyshev inequalities** to estimate probability bounds of crime occurrences.  
- Used **Bayesian inference** to compute conditional probabilities of crime types.  
- Identified high-risk regions and temporal crime trends.  
- Achieved **~15% improvement in prediction accuracy** through statistical modeling.  
- Enabled **~20% improvement in targeted intervention strategies** through insights.  

---

## File Structure

- `probs.ipynb`: Jupyter Notebook containing full probabilistic analysis and visualizations.  
- `probs.py` *(optional)*: Script version of the analysis for reproducibility and cleaner execution.  

---

## Analysis

The project begins with **data preprocessing**, including cleaning and formatting the dataset to ensure consistency.

It then applies:

- **Markov Inequality** to estimate upper bounds of crime probabilities  
- **Chebyshev Inequality** to analyze variance and distribution spread  
- **Bayes' Theorem** to compute conditional probabilities of specific crimes  

Visualization techniques using **Matplotlib** and **Seaborn** are used to:

- Display probability distributions  
- Identify trends over time  
- Highlight high-risk regions  

These analyses provide meaningful insights into crime patterns and support **data-driven decision-making**.
